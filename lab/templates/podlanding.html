About this Lab

When we embarked on developing this lab we realized that understanding Cisco Container Platform and Kubernetes without having some basis of containers didn't make sense. So we developed in the lab a structure that goes through showing some of the container basics. We chose to run the lab on Application Centric Infrastructure (ACI) and HyperFlex (HX) as they were designed for multicloud. CCP can run on any network or compute infrastructure.

The lab from a high level has the following steps:

{{ html_for_step_status|safe }}

We take the student through the following flow of learning:

  • Intro Lab Section: Docker, Docker Networking and Kubernetes (K8s)
    • This section will walk you through a deep-dive on docker
    • You will build, manage and deploy a Webserver container
    • You will also get a deeper understanding of container networking
    • Finally, you will install a kubernetes cluster from scratch!
  • Deploying Cisco Container Platform on Cisco Hyperflex
    • Overview of the platform, components, features etc.
    • Deploying a tenant cluster integrated with Cisco ACI (Applicaton Centric Infrastructure)
  • Deploying an Application on CCP (Cisco Container Platform)
    • Deploy a microservices based Application
    • ACI contracts between Kubernetes and Bare Metal/VM
  • Deploying and managing HX Persistent Storage
    • Deploy a storage integrated Application
    • Manage/upgrade the Application
  • Challenge Labs!!!
