Creating ACI Tenant manually

Cisco Cloud Platform (CCP) using the CNI plugin is able to automatically build a tenant and configure the Kubernetes application profile. In this section, we'll manually build a tenant and expore the Kubernetes integration components. We'll in a later step remove this configuration and monitor how CCP automates this process.

For this lab we ask you to be cautious when working with Tenants to ensure you don't affect other student labs.
